Nikolang

A Nikolang é a linguagem de programação mais bosta já inventada.
Deve ser utilizada apenas por quem escreve código realmente merda.

Entrada
Saída
{{stdout}}
Baixar essa porra

FAZENDO MERDA

Início do código

DEIXA EU TRABALHAR
//código aqui
PORRA

Tipos de dados

BOBO FR = 'a'; (char)
TRISTEZA M1 = 13; (short)
BOSTA M2 = 37; (int)
MERDA M3 = 666; (long)

Tipos de dados

DESGRAÇA T = 0.13; (float)
MUITO DESGRAÇA TD = 0.37; (double)
TCHURURU BOBO BF = 200; (unsigned)

Printar na tela


OLHA ESSA MERDA ("Hello, Mutante");

Ler da tela

BOSTA X;
OLHA ESSA MERDA ("%d", &X);

If


COMO ASSIM CE FALA? (3 > 2)
//código a ser executado
PORRA

If/Else

COMO ASSIM CE FALA? (X > 2)
//caso verdadeiro
CE TA USANDO UMA DROGA?
//caso falso
PORRA

Else if

COMO ASSIM CE FALA? (X > 2)
//X > 2
CE TA BURRO? (X < 2)
//X < 2
CE TA USANDO UMA DROGA?
//X = 2
PORRA

While


BOSTA X = 5;
REUNIÃO DE NOVO (X > 2)
//rodar código
X--;
PORRA

For



BOSTA M;
MAIS UMA REUNIÃO (M = 0; M < 5; M++)
OLHA ESSA MERDA ("%d", M);
PORRA

Break/Continue

BOSTA M;

//continue

//break
NINGUÉM QUER ISSO;
PORRA

Declarar função

OH OS VÉI PELADO (BOSTA NOMEFUNC(BOSTA A, BOSTA B))
//código da função

PORRA

Chamar função

BOSTA A = 5;
BOSTA B = 8;
BOSTA C = SOU JESUS SOMAR(A, B);

Exemplo

Fiz um código bosta! Nada mais justo.

OH OS VÉI PELADO (BOSTA SOMAR(BOSTA A, BOSTA B))
      BORA TRABALHAR A + B;
PORRA

DEIXA EU TRABALHAR
      BOSTA A, B, RES;
      OLHA ESSA MERDA ("Fala, caralho!!\n");
      OLHA ESSA MERDA ("%d %d", &A, &B);
      RES = SOU JESUS SOMAR(A, B);
      OLHA ESSA MERDA ("Oh o resultado ai po: %d\n", RES);
      COMO ASSIM CE FALA? (RES == 37)
            OLHA ESSA MERDA ("Nada mais justo!\n");
      CE TA USANDO UMA DROGA?
            OLHA ESSA MERDA ("Vou arrancar os braços dele!\n");
      PORRA
      BORA TRABALHAR 0;
PORRA
